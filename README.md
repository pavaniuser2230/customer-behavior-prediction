##  Project Overview

This project focuses on analyzing customer behavior using a real-world e-commerce dataset.
The goal is to predict customer satisfaction based on purchase, delivery, and payment details.

---

##  Objectives

* Analyze customer purchasing behavior
* Perform data preprocessing and feature engineering
* Build machine learning models
* Compare model performance
* Identify the best model

---

## Dataset

* Olist Brazilian E-commerce Dataset
* Contains multiple CSV files:

  * Customers
  * Orders
  * Products
  * Payments
  * Reviews
  * Sellers

---

## Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

##  Machine Learning Models

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Random Forest

---

## Results

* All models were evaluated using accuracy score
* Random Forest achieved the highest performance
* Successfully predicted customer satisfaction
